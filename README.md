# Nathaniel — Personal Portfolio

This is a responsive single-page personal portfolio built with:

- **HTML5** (semantic structure)
- **Bootstrap 5** (layout & components)
- **Custom CSS** (`style.css`) — overrides Bootstrap defaults and adds a custom theme
- **Vanilla JS** (`script.js`) — small enhancements
- **Deployment**: GitHub Pages

## Features
- Hero, About, Skills, Projects, Contact sections (single-page, smooth scrolling)
- Responsive design for mobile, tablet, and desktop
- Projects gallery with links to code and demos
- Contact form (can be wired to Formspree or other service)
- Accessible semantic HTML5 tags and focus outlines for keyboard users
